## Getting Started

### Prerequisites
To get started with the Empress Application Template, you need to have a basic understanding of Empress applications.

### Installation
Follow these steps to set up a development environment:

1. Clone the repository:
```sh
git clone https://github.com/empress-eco/application_template.git
```
2. Install the application template:
```sh
bench get-app application_template 
bench install-app application_template
```
For additional details, refer to [Empress Installation Guide](https://github.com/Empress/Empress/wiki/The-Hitchhiker%27s-Guide-to-Installing-Empress-on-Linux).

### Usage
Use the Empress Application Template to expedite your development process. Run tests with the following command:

```sh
bench run-tests --app application_template
```
For more details, refer to [Empress Unit Testing Guide](https://Empress.io/docs/user/en/guides/automated-testing/unit-testing).
